# LendBook 1.0

Bem-vindo ao LendBook 1.0, um sistema de controle de empréstimos em bibliotecas desenvolvido em Java!

## Visão Geral
O LendBook 1.0 é uma aplicação desktop projetada para facilitar o gerenciamento de empréstimos em bibliotecas. Com uma interface intuitiva e funcionalidades robustas, o LendBook 1.0 oferece uma solução eficiente para bibliotecários e usuários.

## Recursos Principais
- **Banco de Dados em JSON:** Utilizamos arquivos JSON para armazenar todos os dados, garantindo facilidade de manipulação.
- **Painéis de Administração e Usuário:** Painéis distintos para administradores e usuários, com ferramentas específicas para cada tipo de usuário.
- **Autenticação de Usuários:** Sistema de login seguro para autenticar usuários e garantir a segurança dos dados.
- **Redefinição de Senha:** Recurso para que os usuários possam redefinir suas senhas de forma segura.
- **Registro de Novos Usuários:** Apenas administradores podem registrar novos usuários, garantindo controle sobre o acesso ao sistema.
- **Extensa Segurança:** Implementamos medidas de segurança robustas para proteger o programa contra qualquer tentativa de comprometimento.

## Como Usar
1. Faça o download do código-fonte do LendBook 1.0.
2. Abra o projeto no NetBeans.
3. Execute o programa e explore as diferentes funcionalidades disponíveis.
4. Sinta-se à vontade para fazer modificações e contribuições para melhorar o LendBook 1.0.

## Arquivos Principais
- **Main.java:** Ponto de entrada do programa.
- **Books.json:** Arquivo JSON para manipulação de dados relacionados a livros.
- **User.json:** Arquivo JSON para gerenciamento de usuários e autenticação.

## Dependências
Este projeto requer a biblioteca json-simple-1.1.1.jar para funcionar corretamente.

## Recomendações de Versões Java
Recomendamos o uso de Java 18.0.2.1 e Java HotSpot(TM) 64-Bit Server VM 18.0.2.1+1-1.

## Execução do Projeto
Para executar o projeto a partir da linha de comando, vá para a pasta dist e digite o seguinte:

java -jar "Biblioteca_V_1_1.jar"
